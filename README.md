# shopping-cart-microservice

## Configuración del Backend
```
 docker compose up
```
```
cd shopping-cart-backend
```
```
pip install -r requirements.txt
```
```
uvicorn main:app --reload
```

## Configuración del Frontend
```
cd ..

```
```
cd shopping-cart-frontend
```
```
npm install
```
```
npm run dev
```
